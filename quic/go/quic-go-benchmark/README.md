Usage of ./quic-client:
  -port int
        服务器端口 (default 8888)
  -rate int
        发送速率限制(Mbps), 0表示无限制
  -server string
        服务器地址 (default "localhost")
  -size int
        数据包大小(字节) (default 1400)
  -time int
        测试持续时间(秒) (default 10)

Usage of ./quic-server:
  -port int
        服务器监听端口 (default 8888)
